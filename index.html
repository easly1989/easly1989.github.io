Nutreek

A self-hosted weekly nutrition planner: build meal plans and track calories and macros (protein, carbs, fat) per meal, day and week, backed by an offline OpenFoodFacts ingredient database. Adaptive TDEE, weight and hydration tracking, workouts and a coach mode for managing clients. One Go binary or one Docker container.

Weekly meal planning Macros & adaptive TDEE Self-hosted · Docker Coach mode

Cupid

A web management system for matchmaking & dating agencies — a cloud-ready port of the classic “Cuore” desktop app. Client records, partner preferences, affinity matching, meetings, contracts and print-ready sheets, with email/Telegram notifications, GDPR tools and encrypted cloud backups. Ships as a single Docker container.
